Matsuda's Portfolio

ポートフォリオサイトです。自慢ポイントは、AWSにサーバーレス構成で載せていること。 EC2は使わずにS3のwebホスティング機能で公開し、お問い合わせフォームもAPI Gateway + lambda + SES で実現しています。 フロント言語にはあまり慣れていないのですが、楽しく制作できました。

使用言語、開発環境等

HTML, CSS, Sass, Javascript, JQuery, AWS(S3, APIGateway, lambda, SES)

制作期間

約2週間

パスジェネ

パスワード自動生成アプリです。
生成だけでなく、データベースへの保存機能や、クリップボードへのコピー機能なども付けました。
保存時にはパスワードをアプリ側(フロント)で暗号化していたり、バック側の処理はajax経由で、
APIGateway + Lambda + DynamoDB といったサーバーレス構成となっているところがアピールポイントです。

使用言語、開発環境等

HTML, CSS, Javascript, AWS(S3/APIGateway/Lambda/DynamoDB), VirtualBox(Ubuntu)

制作期間

約1ヶ月半ほど

センスオブセンス

MBSD情報セキュリティコンテストに、4人チームのリーダーとして参加した時の作品です。
情報セキュリティに関する教育コンテンツを作成せよ、とのお題に対し、作品のコンセプトは「すべての人をセキュリティ人材へ」。
ボイスロイドを使用した動画講座による学習と、ノベルゲーム形式での演習からなる学習コンテンツを制作しました。
結果は全国7位でしたが、作品のみの点数なら全国2位。
松田の担当作業は、プロフェクトのマネジメント、ノベルゲームの文章執筆、ボイスロイドの調声、動画の制作、他諸々。

使用言語、開発環境等

HTML, CSS, Sass, Javascript, irohaboard, AWS(EC2), Voiceroid2, aviUtl

制作期間

約2ヶ月

※テスト用に、動画講座の一部を載せております。
音が出ますのでご注意ください。
